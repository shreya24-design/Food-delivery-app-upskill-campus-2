Food Delivery App

Project Description

This is a Full Stack Food Delivery Application developed during the internship program under upskill Campus and UniConverge Technologies Pvt Ltd (UCT).

The application allows users to browse food items, add products to cart, and place food orders through a user-friendly interface.

Features

- User Login and Registration
- Browse Food Items
- Add to Cart
- Order Placement
- Responsive User Interface

Technologies Used

- HTML
- CSS
- JavaScript
- Full Stack Web Development
